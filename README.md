# Svelte GitHub Page Starter

A svelte start template for GitHub pages.

## Getting Started

You can fork this repository or simply use [degit](https://github.com/Rich-Harris/degit).

## Libraries & Tools

- Svelte
- Vite
- pnpm
- gh-pages

## Development

```sh
pnpm dev
```

## Deploy to GitHub Pages

```sh
pnpm deploy
```
